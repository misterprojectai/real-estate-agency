# Seller Transaction Checklist

last_updated: 2025-05-14
version: 1.0
jurisdiction: Texas residential — TREC contracts
used_by: 04_transaction_coordinator (seller deals only)

---

When creating a Deal File, TC copies applicable items from this checklist
and tracks completion status. Note this version in the Deal File.

Update this file and increment version when TREC requirements change or
Austin market practice shifts. Active deals retain the version they were
created under.

---

## At listing

- [ ] Seller's Disclosure Notice completed and signed
- [ ] Lead-based paint disclosure completed and signed (if pre-1978 construction)
- [ ] Existing survey located (or new survey ordered)
- [ ] HOA resale certificate ordered (if HOA present)
- [ ] Listing agreement signed
- [ ] MLS listing reviewed and approved by seller

## At contract execution

- [ ] TREC One to Four Family Residential Contract — fully executed
- [ ] Earnest money delivery to title confirmed
- [ ] Option fee receipt confirmed
- [ ] Addendum for Property Subject to Mandatory Membership in HOA — signed (if HOA)
- [ ] Any additional addenda identified and signed
- [ ] Seller's Disclosure Notice delivered to buyer

## Option period

- [ ] Buyer inspection confirmed scheduled
- [ ] Inspection completed
- [ ] Inspection response — repair amendment or rejection decision made
- [ ] Amendment to Contract signed (if repairs negotiated)
- [ ] Option period termination confirmed in writing

## Title

- [ ] Title company opened order
- [ ] Title commitment received
- [ ] Title commitment reviewed — existing liens identified
- [ ] Payoff statements ordered from all lenders/lienholders
- [ ] Payoff statements received — amounts confirmed
- [ ] Tax certificates ordered
- [ ] Any curative title work completed

## HOA (if applicable)

- [ ] HOA resale certificate delivered to buyer within deadline
- [ ] HOA transfer fee confirmed and allocated on settlement statement
- [ ] HOA transfer documentation completed
- [ ] HOA transfer confirmation received

## Pre-closing

- [ ] Final walkthrough by buyer completed
- [ ] No undisclosed changes to property confirmed
- [ ] Settlement statement reviewed — net proceeds match expectations
- [ ] Wire instructions provided to title company — delivery confirmed
- [ ] Utility transfer dates confirmed with buyer
- [ ] All personal property to be removed confirmed cleared

## Closing

- [ ] ALTA Settlement Statement reviewed before signing
- [ ] Deed signed and notarized
- [ ] All documents signed
- [ ] Deed recorded — confirmation obtained
- [ ] Net proceeds wire confirmed received
- [ ] Keys, garage remotes, mailbox keys, and access codes transferred to buyer
- [ ] Title policy issued to buyer

## Post-close

- [ ] Client follow-up sent
- [ ] Lead Card status updated to closed
- [ ] Referral request noted in activity log
- [ ] If buying next: Lead Card direction updated, buyer profile activated
